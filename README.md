# Rahul Sharma Portfolio

## Run locally

Open `index.html` in your browser.

## Replace profile image

Replace `assets/profile-photo.svg` with your own photo and keep the same file name, or update the image path in `index.html`.

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Push this folder contents to the `main` branch.
3. Go to `Settings > Pages`.
4. In `Build and deployment`, set `Source` to `Deploy from a branch`.
5. Select branch `main` and folder `/ (root)`, then save.
6. Your site will be available at `https://<your-username>.github.io/<repo-name>/`.

## Deploy on Netlify

1. Sign in to Netlify.
2. Click `Add new site` -> `Import an existing project` (or drag-drop this folder).
3. Build command: none.
4. Publish directory: `.`
5. Deploy.
